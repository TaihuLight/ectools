Welcome

Welcome to the official ectop documentation. Here you will find a detailed view of all ectop can do.

This monitoring application is a top like monitor, where you can easily add/remove sensors (physical measures or estimators).

Dependencies

  • kernel source v2.6 (don't work with 3.0+) [download]

Adding/removing columns into the interface

This example show how to add or remove columns with sensors into the main.cpp file.

XML monitor loaded

void
addSensors(MonitorEctop* m)
{
m->addSensor(new SensorCpuUsage());
m->addSensor(new PerfCount(PERF_TYPE_HARDWARE, PERF_COUNT_HW_CPU_CYCLES));
m->addSensor(new PerfCount(PERF_TYPE_HARDWARE, PERF_COUNT_HW_CACHE_MISSES));
m->addSensor(new LinearRegression(110, 220, new SensorCpuUsage()));
m->addSensor(new LinearRegression());
}
