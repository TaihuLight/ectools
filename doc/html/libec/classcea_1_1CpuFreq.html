Detailed Description

Gets CPU frequency.

This sensor collects the current CPU frequency in KHz from the /sys/devices/system/cpu/cpuID/cpufreq/cpuinfo_cur_freq virtual file. To access this file, one need to have administrative permissions. This sensor has the latency as described into the cpuinfo_transition_latency file.
