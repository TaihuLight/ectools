Detailed Description

template<int TAG = -1>
class cea::GridModel::col_iterator< TAG >

Column "tag" Iterator.

This iterator allow to select only certain item follow their tag.

By default the iterator .

If all rows/columns need to be updated then use classic iterator of GridModel::columns or GridModel::rows member.

Example

for(GridModel::col_iterator<CPU> c; c != 0; ++c) {
// c-> or *c == GridColumn&
}
