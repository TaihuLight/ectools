#define CEA_VALUE_FAST_MODE   0

Dynamic memory allocation and deletion mode for value.

If set to 1 Then avoid dynamic memory allocation and deletion Else use dynamic memory to store value.

When dynamic memory mode is disabled this class run faster but take more memory space since memory space is reserved.
