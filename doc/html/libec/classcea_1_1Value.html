Output on std::ostream the value.

This function allow to use value with out stream

Example

// Value to show
Value v = 23.5;
// Show the value
std::cout << "Value = " << v << std::endl;
// Change the value type
v.setType(Value::STRING);
// Set a new value
v = "text to show";
// Show the value
std::cout << "Value = " << v << std::endl;
// Screen result
// Value = 23.5
// Value = text to show
