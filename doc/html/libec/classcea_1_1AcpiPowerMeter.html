Detailed Description

A power meter based on ACPI information This PowerMeter uses the actual current and voltage information from ACPI enabled hardware. It checks for the reading access of the following files:

  • Voltage (v in uV): /sys/class/power_supply/BAT0/voltage_now
  • Current (c in uA): /sys/class/power_supply/BAT0/current_now
