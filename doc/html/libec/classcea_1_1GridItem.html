unsigned int cea::GridItem::getUpdateCount ( ) const

Get the current update counter value.

This function is usefull to know if the row or column have changed.

Example

// A Row
GridRow& row;
// Store in memory the return of this function
unsigned int storedRowUpdateCount = row.getUpdateCount();
// Execute piece of code
// ...
// ...
// Check if Row have changed
if (row.getUpdateCount() != storedRowUpdateCount) {
// Row changed
}else{
// Row not changed
}

GridItem& cea::GridItem::setArg ( unsigned int  id,
const Value &  v 
)

Associate to an arg id a value.

Arg are usefull system to store extra value for a row or column. The value are stored as a map identified by an unsigned int.

By this system its possible to store an infinity of arg value of any type for a row or column.

Remark: By default the args take no memory until the first usage where an map is created in memory.

Example

// Create a model
GridModel m;
// Add 4 CPU Core Column and use an arg to distinct core
for(unsigned int i = 0; i < 4; i++) {
m.addColumn("Core", Value::INT)
.setArg(0, i) // Set Core ID in arg 0
.setArg(10, "Core") // Set Core name in arg 10
;
}
