Detailed Description

Base class for grid view.

By default this class is a fully functional view which simply output all visible rows and columns in the std::cout ostream.

There are 2 mode of view:

  • (default) Render mode – Render function is used to render item.
  • Event mode – Each time a change is done on view a specific event is raised.
    (This mode is provide for GUI where almost time no render is needed just "object" state change)

Map a filter with a style.

Style-filter allow to change style of some row depend on their cells value.

Example

Set all pid > 100 to red

// A grid view
GridView view;
// Create style and set the style
view.setStyle(10, GridStyle().setColor(Color::RED));
// Add Style filter
view.addStyleFilter("PID > 100", 10);

bool cea::GridView::needRender ( ) const

Check if render needed.

If there is no change on the view or on the cells
Then this function will return false.

Example

// A Grid Model
GridModel model;
// A Grid View
GridView view;
// Infinite loop
while (1) {
// GridModel change ...
// Render only if it's needed
if (view.needRender()) {
view.render();
}
}
Returns
