Detailed Description

Terminal grid controller.

Action possible

  • Sort – Default action. Repeat of this action on a column will invert order of Sort (ASCENDING or DESCENDING).
  • Sum – Activate or Desactivate the sum computation.
    Invert status each time action raised.
  • Fix – Fix or unfix a column.
    Invert status each time action raised.

Key interpreted

  • 0 – Disable sort
  • 1 to 9 – Action on column 1 to 9.
    With 1 egal to the first column visible in screen.
    Action change into Sort.
  • s – Change Action into Sum.
  • f – Change Action into Fix.
