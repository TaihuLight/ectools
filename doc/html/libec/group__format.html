enum cea::FormatBlockType

Type of block formatted (mainly used for log system)

Allow to all formatter to select what block name use for an add,update or remove entry.

Example

For a list of entries 2 added item, 1 removed, 3 updated:

In CSV default block name are (+ for add, - for remove, . for update). CSV formatted Result will look like:

+,Item1
-,Item2
+,Item3
.,Item4
.,Item5
.,Item6

In XML default block name are ("add", "delete", "update") XML formatted result will look like:

<add data="Item1" />
<delete data="Item2" />
<add data="Item3" />
<update data="Item4" />
<update data="Item5" />
<update data="Item6" />
See Also
cea::CSVFormatter, cea::XMLFormatter
