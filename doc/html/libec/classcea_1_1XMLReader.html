Detailed Description

XML DOM parser able to read standard XML format.

Balise path format

The balise path is a little XPath format, in fact only separator can be used to search a balise, no array or XPath function can be used.
Basically the balise path consist of Balise::name separated by a char which can be '/', '\', '|' or ';'.
Exemple of balise path to get title of an well-formed html page : "html/head/title". 

@section cea_XMLReader_attrName Attribute format
The attribute take care of case and respect the XML attribute format.
