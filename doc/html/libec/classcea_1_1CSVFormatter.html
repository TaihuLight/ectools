virtual void cea::CSVFormatter::formatComment ( const std::string &  comment)
virtual

Format a comment in CSV.

As CSV format have no grammar rule for comment, the implementation chosen is to add a line with protected string value.
Format of the comment is "#comment" (the char " is important).

This implementation fit perfectly with Spreadsheet Software.
