virtual void cea::GridCtrl::handleEvent ( const Event &  e)
virtual

Handle an event.

This function handle any event Terminal, GUI or other.
By default this function do nothing.
When a new controller is created this function need to be overridden in order to interpret event.

For example TermGridCtrl override this function to interpret terminal event.
