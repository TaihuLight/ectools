static pid_t cea::SystemInfo::getPidMax ( )
static

Gets PID's maximum value.

Returns
A pid_t with the PID maximum value Gets the value at which PIDs wrap around (i.e., this value is one greater than the maximum PID). Its default value is 32768. On 32-bit platforms, 32768 is the maximum value for pid_max. On 64-bit systems, pid_max can be set to any value up to 2^22 (PID_MAX_LIMIT, approximately 4 million).
